# study_rails_05

## blog_bootstrap

### command

#### setup

- mkdir blog_bootstrap
- cd blog_bootstrap
- bundle install --path vendor/bundle
- bundle exec rails new . -B --skip-test --skip-turbolink

#### example

- rails g scaffold article title:string
- bundle exec rails action_text:install
